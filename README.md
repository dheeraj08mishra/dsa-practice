Daily DSA practice — JavaScript (VS Code) and Java (IntelliJ). One problem a day, logged here automatically.

## Structure

```
javascript/
  arrays/
  strings/
  linked-list/
  trees/
  ...
java/
  arrays/
  strings/
  linked-list/
  trees/
  ...
```

## How I commit

After solving a problem and saving the file, run:

```
./scripts/dsa-commit.sh
```

This auto-detects the new/changed file, writes the commit message, updates the table below, and pushes.
